UK Job Market Analysis for Data Roles – 2025

Live UK job data visualized through API-powered dashboards

📌 Project Summary

Aspiring data professionals in the UK often struggle to identify current job trends, industry demands, and competitive salary ranges. This Power BI dashboard was built to analyze live job postings using API data, helping candidates and recruiters understand key hiring trends across locations, industries, and contract types.

🛠 Tools Used

  • Python
  • Pandas
  • REST API (Adzuna)
  • Power BI
